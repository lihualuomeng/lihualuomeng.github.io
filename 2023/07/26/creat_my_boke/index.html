creat_my_boke

2023/7/26

准备环境

1、安装node.js：https://nodejs.org/en（左边的那个）
2、安装Git：https://git-scm.com/downloads
进入页面后：找到windows，点击打开，找到“32-bit Git for Windows Setup“（32位）或“64-bit Git for Windows Setup“（64位），根据自己的操作系统安装
3、注册一个github的账号
注意：github名字最好用英文，长度长一点,指令注意大小不能变，如果打不开网页，可以挂一个代理

操作

1、在本地找一个地方新建文件夹目录，用英文命名，推荐命名为”boke“
2、右击boke，点击git bash here，打开git命令窗口
3、在git命令窗口中输入：
a.npm install -g hexo-cil
b.hexo init blog
c.cd blog
d.hexo g

效果图如下：

5、在git命令窗口中输入cd blog，如图：

再输入以下几条命令检查网络雏形是否可行

（1）输入“hexo new test_my_site” //创建新文章

（2）输入“hexo g” //生成

（3）输入“hexo s”//启动服务器预览

当出现以下结果：

即可用输入127.0.0.1：4000进行预览，查看网页发现结果有刚创立的“test_my_site”，代表网站创立成功，但还无法做到永久网页访问，还要进行以下步骤

6、配置ssh密钥

（1）输入“cd ~/.ssh”指令来检查本机已存在的密钥

观察输入后提示，如果输入后，出现如图：

说明你创建过ssh

如果出现“No such file or directory”,说明你是第一次使用git

（2）第一次使用则需要输入“ssh-keygen -t rsa -C”自己注册gitthub的邮件地址””

然后进行三次回车，最终会生成一个.ssh文件，其一般在c盘的用户文件中，当前计算机用户名字命名的文件夹中，打开.ssh\id_rsa.pub文件，记事本打开并复制其中的内容，打开你的github主页

由于github是全英文，建议装个翻译插件，没装翻译插件就请跟着以下步骤跟我走：

第一步：点击头像

第二步：找到个人设置并点击

第三步：打开SSH和GPG密钥

第四步：点击new ssh key按钮，在key中输入在.ssh\id_rsa.pub复制的内容

，再点击Add SSH key，即可添加成功

（3）尝试SSH连接，测试是否成功

输入命令“ssh -T git@github.com”#照着这个命令输入即可

如果提示“Are you sure you want to continue connecting (yes/no)?”

输入“yes”

如果你看到了下面的这个，就算成功了

“Hi ‘你自己的github账户名’! You’ve successfully authenticated, but GitHub does not provide shell access.”

（4）在创建的目录（blog目录下）下打开cmd（以管理员的身份）

输入“npm install hexo-deployer-git -save”

（5）在git命令窗口中

a.输入“git config –global user.name “你的github用户名””

b.输入“git config –global user.email “你的github注册邮箱””

7、准备上传代码

（1）在github上新建一个数据库，命名为“你的github的用户名.github.io”

a.点击头像

b.点击“your repositories”

c.找到并点击“new按钮”

d.在Repository name 存储库名称一栏输入“你的github的用户名.github.io”，然后划到最底下找到“create repository”即可创立
